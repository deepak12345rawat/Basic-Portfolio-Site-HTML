<DOCTYPE>

This DOCTYPE tag defines the type of document and in which version of html we are working..

<html>

The html tag is used to start a documents and end it at end of the document.

<head>

This tag contains the metadata and internel CSS is used in this tag.

<title>

This tag defines a title for document and it is shown on top of the screenon a bar.

<body>

This tag tells us about the body of the document.

<h1>to<h6>

This tag defines the html headingsin the increasing order of their importance.

<p>

This tag is used to write the paragraph within the web page.

<br>

This tag is used for line break in the page.

<a>

This page is used to add links in a page to increase it's accessibility.

<img>

This is used for adding image in html page.

<iframe>

This tag is used to add inline frame in page of any size.

<form>

This tag is used to create forms like registration form etc.

<fieldset>

This tag is used to create border lines around a form content.

<legend>

This tag is used for heading of form.

<button>

This tag is used for creating different types of buttons in forms or others also.

<textarea>

It is used for putting additional space for text.

<ul>

It is used to create unorderd list.

<ol>

It is used to create ordered list

<li>

It is used to create list items.

<dl>

It is used to create description lists.

<style>

It is used to put styles in pages with CSS
